# Расчёт рассеивания загрязняющих веществ в атмосфере (ОНД-86)

## Документация проекта

---

## 1. Описание продукта

### 1.1 Назначение

Веб-приложение для расчёта рассеивания загрязняющих веществ (ЗВ) в приземном слое атмосферы по методике ОНД-86 (Общесоюзный нормативный документ, утверждённый Приказом МПР №273).

Приложение заменяет устаревшие десктопные программы (такие как «Радуга») и ручную визуализацию в CorelDRAW, объединяя расчёт, визуализацию и формирование отчётов в одном инструменте.

### 1.2 Целевая аудитория

- Инженеры-экологи
- Специалисты по охране атмосферного воздуха
- Проектные организации, выполняющие разделы ОВОС (оценка воздействия на окружающую среду)
- Руководители экологических отделов (просмотр результатов)

### 1.3 Проблемы, которые решает приложение

| Проблема (текущее состояние) | Решение |
|---|---|
| «Радуга» даёт только таблицы с цифрами, без визуализации | Результаты отображаются на интерактивной карте (heatmap, сетка с цифрами, изолинии) |
| Ручной перенос данных из «Радуги» в CorelDRAW для создания карт рассеивания | Автоматическая визуализация на спутниковой/картографической подложке |
| Ручной ввод большого объёма данных | Справочники городов, веществ и ПДК — выбор из списка |
| Нет возможности быстро менять параметры и видеть результат | Интерактивный ввод с мгновенным пересчётом |
| Сложно формировать отчёты | Автоматическая генерация PDF-отчёта с таблицами и графиками |

### 1.4 Ключевые преимущества

- Работает в браузере — не требует установки
- Расчёт + визуализация + отчёт в одном месте
- Интерактивная карта с маркерами источников (drag & drop)
- Поддержка нескольких источников выбросов одновременно (до 400 в целевой версии)
- Загрузка собственной подложки (генплан, ситуационный план)
- Двуязычный интерфейс (русский / узбекский)
- Экспорт PDF-отчёта для согласования

## 2. Функциональные требования

### 2.1 Ввод данных по источникам выбросов

Пользователь задаёт один или несколько точечных источников выбросов со следующими параметрами:

| Параметр | Обозначение | Единица | Описание |
|---|---|---|---|
| Название источника | — | текст | Идентификация источника |
| Высота трубы | H | м | Высота источника над уровнем земли |
| Диаметр устья трубы | D | м | Внутренний диаметр на выходе |
| Скорость выхода газов | w0 | м/с | Скорость газовоздушной смеси на выходе |
| Температура газов | Tг | °C | Температура выбрасываемой смеси |
| Выброс (г/с) | M | г/с | Мощность выброса в граммах в секунду |
| Выброс (т/год) | M год | т/год | Валовый выброс в тоннах в год (для таблиц ПДВ/ОВОС) |
| Координаты | lat, lon | градусы | Позиция на карте (ввод вручную или клик на карте) |

**Ограничения по количеству источников:**

| Версия | Кол-во источников | Способ ввода |
|---|---|---|
| MVP | до 50 | Карточки, табличный ввод, импорт CSV/Excel |
| Целевая | до 400 | + оптимизированный расчёт, группировка по цехам, индикатор прогресса |

**Способы ввода данных:**

**Способ 1 — Карточки (для 1–10 источников):**
- Добавление нового источника кнопкой «+ Добавить»
- Удаление источника
- Размещение источника кликом на карте
- Перетаскивание маркера источника на карте

**Способ 2 — Табличный ввод (для 1–50 источников):**

Переключение в режим таблицы (кнопка «Табличный вид»). Данные вводятся как в Excel — строки = источники, столбцы = параметры:

```
| Название     | H, м | D, м | w0, м/с | Tг, °C | M, г/с | M, т/год | Широта  | Долгота |
|--------------|------|------|---------|--------|--------|----------|---------|---------|
| Труба ТЭЦ-1 | 45   | 1.2  | 12.0    | 180    | 8.5    | 268.06   | 41.2995 | 69.2401 |
| Котельная    | 20   | 0.8  | 6.0     | 120    | 3.2    | 100.93   | 41.3010 | 69.2380 |
| ...          | ...  | ...  | ...     | ...    | ...    | ...      | ...     | ...     |
```

Поддержка:
- Копирование/вставка из Excel (Ctrl+V) — парсинг табличных данных из буфера обмена
- Добавление/удаление строк
- Сортировка по столбцам

**Способ 3 — Импорт из файла (для 10–400 источников):**

Кнопка «Импорт из файла» принимает:

| Формат | Описание |
|---|---|
| CSV (.csv) | Текстовый файл с разделителем (запятая или точка с запятой) |
| Excel (.xlsx) | Стандартный формат Microsoft Excel |

Требования к файлу:
- Первая строка — заголовки столбцов
- Обязательные столбцы: Название, Высота (H), Диаметр (D), Скорость (w0), Температура (Tг), Выброс г/с (M)
- Необязательные столбцы: Широта, Долгота, Выброс т/год (M год)
- Названия столбцов распознаются гибко (допускаются варианты: «Высота», «H», «H, м», «height»)

Кнопка «Скачать шаблон» — пользователь скачивает готовый Excel-шаблон с правильными заголовками и примером заполнения.

После импорта:
- Данные отображаются в табличном виде для проверки
- Ошибки валидации подсвечиваются (пустые обязательные поля, нечисловые значения)
- Пользователь может исправить данные перед расчётом
- Источники без координат размещаются автоматически вокруг центра города (можно скорректировать)

### 2.2 Ввод метеорологических данных

| Параметр | Единица | Описание |
|---|---|---|
| Город / регион | — | Выбор из справочника (определяет коэффициент A и температуры) |
| Скорость ветра | м/с | Скорость ветра на уровне флюгера |
| Направление ветра | градусы | Откуда дует ветер (0° = С, 90° = В, 180° = Ю, 270° = З) |
| Класс устойчивости атмосферы | A–F | По классификации Пасквилла–Гиффорда |
| Температура воздуха | °C | Температура наружного воздуха |

При выборе города автоматически подставляется средняя температура и климатический коэффициент A.

**Автозагрузка метеоданных из интернета (кнопка «Загрузить текущую погоду»):**

При нажатии приложение запрашивает текущие метеоданные по координатам выбранного города через API Open-Meteo (бесплатный, без ключа) и автоматически заполняет поля:

| Параметр | Источник данных |
|---|---|
| Температура воздуха | Open-Meteo: `temperature_2m` |
| Скорость ветра | Open-Meteo: `wind_speed_10m` |
| Направление ветра | Open-Meteo: `wind_direction_10m` |
| Класс устойчивости | Автоматический расчёт по методу Пасквилла–Тёрнера (скорость ветра + облачность + время суток) |

Определение класса устойчивости по Пасквиллу–Тёрнеру:

| Скорость ветра, м/с | День, сильная радиация | День, умеренная | День, слабая / облачность | Ночь, ясно | Ночь, облачно |
|---|---|---|---|---|---|
| < 2 | A | A–B | B | F | F |
| 2–3 | A–B | B | C | E | F |
| 3–5 | B | B–C | C | D | E |
| 5–6 | C | C–D | D | D | D |
| > 6 | C | D | D | D | D |

Пользователь может скорректировать любые автозаполненные значения вручную после загрузки.

Примечание: для работы автозагрузки требуется доступ к интернету. При отсутствии связи все поля заполняются вручную как прежде.

**Режим направления ветра (переключатель):**
- «360° — все направления» (по умолчанию) — рассеивание во все стороны, наихудший сценарий
- «Одно направление» — факел только по указанному направлению ветра

### 2.3 Выбор загрязняющего вещества

Пользователь выбирает вещество из встроенного справочника или добавляет своё:

**Выбор из справочника:**
- Выпадающий список с поиском (код, название)
- При выборе автоматически подставляется ПДК м.р. и класс опасности

**Добавление пользовательского вещества (кнопка «+ Добавить вещество»):**

Открывается форма с полями:

| Поле | Обязательное | Описание |
|---|---|---|
| Название вещества | Да | Произвольное название |
| Код вещества | Нет | Код по классификатору (если известен) |
| ПДК м.р., мг/м3 | Да | Максимально разовая ПДК (используется в расчёте) |
| ПДК с.с., мг/м3 | Нет | Среднесуточная ПДК (для справки в отчёте) |
| Класс опасности | Нет | 1, 2, 3 или 4 |

Добавленное вещество сохраняется в локальном списке пользователя (localStorage) и доступно в последующих сессиях. Пользовательские вещества отображаются в справочнике отдельной группой «Мои вещества» и могут быть отредактированы или удалены.

### 2.4 Параметры расчётной области

| Параметр | Единица | Описание |
|---|---|---|
| Радиус расчёта | м | Расстояние от центра (центроида источников) до границы сетки |
| Шаг сетки | м | Расстояние между точками расчёта |
| ПДК вещества | мг/м3 | Предельно допустимая концентрация для сравнения |

### 2.5 Расчёт

Расчёт выполняется по методике ОНД-86:

1. **Для каждого источника** вычисляются:
   - Максимальная приземная концентрация Cm (мг/м3)
   - Расстояние до максимума Xm (м)
   - Тип источника (горячий / холодный)
   - Промежуточные коэффициенты (f, vm, m, n, V1)

**ВАЖНО: Режимы расчёта по направлению ветра:**

| Режим | Описание |
|---|---|
| Одно направление | Расчёт для заданного направления ветра (факел по ветру) |
| Полный обзор (360°) | Расчёт для всех направлений ветра с шагом 10° (36 направлений). В каждой точке сетки берётся **максимальная** концентрация из всех направлений. Это стандартный режим для ОВОС — показывает наихудший сценарий |

Режим 360° используется по умолчанию. Пользователь может переключиться на расчёт для одного направления ветра.

При режиме 360° карта рассеивания показывает концентрации **вокруг источника во все стороны**, а не только по направлению ветра. Это соответствует требованиям нормативной документации для определения санитарно-защитной зоны (СЗЗ).

2. **На регулярной сетке** вычисляются:
   - Нормированная осевая функция s1(r), где r = x/Xm
   - Поперечное гауссово распределение s2(y, x)
   - Суммарная концентрация от всех источников в каждой точке

3. **Итог:**
   - Поле приземных концентраций (мг/м3)
   - Координаты и значение максимума
   - Сравнение с ПДК (превышение / норма)

### 2.6 Визуализация результатов

Три режима отображения на карте:

| Режим | Описание |
|---|---|
| Heatmap | Цветовое поле концентраций (градиент от синего к красному) |
| Сетка ОНД | Ячейки с числовыми значениями концентраций, жёлтые метки осей X/Y в метрах |
| Изолинии (планируется) | Линии равных концентраций с подписями значений |

Дополнительно:
- Маркер максимума концентрации (красная звезда)
- Popup с информацией при клике на маркер
- Переключение подложки: OSM / Спутник / Без карты
- Загрузка пользовательской подложки (генплан, ситуационный план)

### 2.7 Панель результатов

- Индикатор превышения / непревышения ПДК
- Максимальная концентрация (мг/м3)
- Отношение Cmax / ПДК
- Координаты точки максимума
- Таблица результатов по каждому источнику (Cm, Xm)

### 2.8 Автоформирование стандартных таблиц (ПДВ + ОВОС)

После ввода данных по источникам приложение автоматически формирует стандартные таблицы, которые используются в разделах проектной документации. Таблицы доступны для просмотра в интерфейсе и экспорта в PDF/Excel.

**Таблицы раздела ПДВ (проект предельно допустимых выбросов):**

| № | Таблица | Содержание |
|---|---|---|
| П-1 | Инвентаризация источников выбросов | Перечень всех источников: номер, название, координаты, H, D, w0, Tг, V1 |
| П-2 | Параметры выбросов ЗВ по источникам | Источник → вещество → выброс (г/с, т/год) |
| П-3 | Суммарные выбросы по предприятию | Вещество → суммарный выброс (г/с, т/год) от всех источников |
| П-4 | Результаты расчёта рассеивания | Вещество → Cmax (мг/м3) → ПДК → Cmax/ПДК → превышение (да/нет) |
| П-5 | Предлагаемые нормативы ПДВ | Источник → вещество → ПДВ (г/с, т/год) |

**Таблицы раздела ОВОС (охрана атмосферного воздуха):**

| № | Таблица | Содержание |
|---|---|---|
| О-1 | Характеристика источников выбросов | H, D, w0, Tг, V1, тип (организованный/неорганизованный) |
| О-2 | Перечень и количество ЗВ | Код, название, класс опасности, ПДК м.р., ПДК с.с., выброс (г/с, т/год) |
| О-3 | Метеорологические условия расчёта | Город, коэффициент A, класс устойчивости, скорость и направление ветра, температура |
| О-4 | Результаты расчёта приземных концентраций | Вещество → фон → расчётная Cmax → Cmax + фон → ПДК → доля ПДК |
| О-5 | Вклад отдельных источников | Источник → вещество → Cm (мг/м3) → Xm (м) → доля от общего максимума (%) |
| О-6 | Сводная таблица воздействия | Вещество → класс опасности → ПДК → Cmax → превышение → вывод |

**Формирование таблиц:**
- Таблицы заполняются автоматически из введённых данных и результатов расчёта
- Просмотр в интерфейсе: отдельная вкладка «Таблицы ПДВ/ОВОС» в панели результатов
- Экспорт: PDF (для вставки в отчёт) и Excel (для дальнейшего редактирования)
- Пользователь может добавить название предприятия, адрес, номер проекта — эти данные подставляются в заголовки таблиц

### 2.9 Экспорт отчётов

Приложение формирует два типа экспорта:

**PDF-отчёт (полный):**

1. Титульная страница (название предприятия, дата, номер проекта)
2. Таблицы ПДВ (П-1 — П-5)
3. Таблицы ОВОС (О-1 — О-6)
4. Метеорологические условия расчёта
5. Карта-схема рассеивания (график поля концентраций)
6. Заключение (превышение / норма по каждому веществу)

**Excel-экспорт:**
- Каждая таблица — отдельный лист в файле .xlsx
- Данные доступны для редактирования и вставки в собственные шаблоны

### 2.10 Карточка предприятия

В верхней части приложения (или в отдельной вкладке «Проект») пользователь заполняет данные предприятия, которые подставляются во все таблицы и отчёты:

| Поле | Обязательное | Пример |
|---|---|---|
| Название предприятия | Да | ООО «Навои Азот» |
| Адрес | Нет | г. Навои, ул. Промышленная, 1 |
| ИНН / СТИР | Нет | 123456789 |
| Номер проекта | Нет | ЭКО-2026-042 |
| Заказчик | Нет | АО «Навоиазот» |
| Разработчик | Нет | ООО «ЭкоПроект» |

Данные сохраняются в localStorage и доступны при следующем открытии. Подставляются в титульную страницу PDF-отчёта и заголовки таблиц ПДВ/ОВОС.

### 2.11 Расчёт границы санитарно-защитной зоны (СЗЗ)

После расчёта рассеивания приложение автоматически определяет границу СЗЗ — контур, на котором концентрация загрязняющего вещества равна ПДК.

**Отображение на карте:**
- Замкнутая линия (полигон) красного цвета — граница СЗЗ по расчёту
- Подпись с расстоянием от источника до границы СЗЗ в каждом направлении
- Заливка зоны превышения ПДК полупрозрачным красным

**В таблицах и отчёте:**
- Максимальное расстояние от источника до границы СЗЗ (м)
- Минимальное расстояние (м)
- Площадь зоны превышения ПДК (га)
- Рекомендация по размеру СЗЗ (с учётом нормативных классов: I — 1000 м, II — 500 м, III — 300 м, IV — 100 м, V — 50 м)

### 2.12 Сравнение сценариев (до/после)

Пользователь может создать два сценария для одного проекта:

| Сценарий | Описание |
|---|---|
| Базовый (до) | Существующее положение — текущие параметры источников |
| С мероприятиями (после) | Изменённые параметры: установка фильтров, увеличение высоты трубы, снижение выбросов |

**Режим сравнения:**
- Карта делится на две половины (split view) или показывает наложение обоих сценариев
- Таблица сравнения: Cmax до → Cmax после → снижение (%) по каждому веществу
- Границы СЗЗ обоих сценариев на одной карте (красная = до, зелёная = после)
- В PDF-отчёте: раздел «Эффективность мероприятий» с таблицей сравнения

### 2.13 Автосохранение проекта

Все введённые данные (источники, метео, вещества, параметры предприятия) автоматически сохраняются в localStorage браузера каждые 30 секунд и при каждом изменении.

- При открытии приложения: автоматическое восстановление последнего сеанса
- Кнопка «Сохранить проект» — скачивание файла .json со всеми данными

### 3.4 Основные формулы ОНД-86

**Максимальная приземная концентрация (горячий источник):**

```
Cm = A * M * F * m * n * eta / (H^2 * (V1 * dT)^(1/3))
```

где:
- A — климатический коэффициент
- M — мощность выброса, г/с
- F — коэффициент оседания (1.0 для газов)
- m, n — безразмерные коэффициенты
- eta — коэффициент рельефа (1.0 для ровной местности)
- H — высота источника, м
- V1 — объёмный расход газов, м3/с
- dT = Tг - Tв — разность температур

**Расстояние до максимума:**

```
Xm = d * H
```

где d — безразмерный коэффициент, зависящий от vm и f.

**Нормированная осевая функция s1:**

```
При r <= 1:  s1 = 3*r^4 / (1 + 2*r^4)
При r > 1:   s1 = 3 / (r^4 + 2)
```

где r = x / Xm.

**Поперечное распределение s2 (гауссово):**

```
s2 = exp(-0.5 * (y / sigma_y)^2)
```

где sigma_y = |x| * sigma_coeff (sigma_coeff зависит от класса устойчивости).
