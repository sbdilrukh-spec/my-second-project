## 1. О компании и деятельности

### 1.1 Профиль

Организация специализируется на разработке экологической документации для промышленных, добывающих, сельскохозяйственных и сервисных предприятий Узбекистана.

### 1.2 Основные направления

| Направление | Описание |
|---|---|
| **ПЗВОС** | Проект заявления о воздействии на окружающую среду |
| **ПДВ** | Предельно допустимые выбросы в атмосферу |
| **ПДО** | Предельно допустимые объёмы образования отходов |
| **ПДС** | Предельно допустимые сбросы в водные объекты |
| **ЗЭП** | Заключение экологической экспертизы проекта |
| **КЭН** | Корпоративные экологические нормативы |
| **РСВ** | Расчёт степени воздействия |
| **ГРР** | Экологическая документация для геологоразведочных работ |
| **УТЭР** | Удельные технико-экономические расчёты |

### 1.3 География и масштаб

- **Регионы**: Навоийская, Самаркандская, Бухарская области, Каракалпакстан
- **Объём работ**: 40+ проектов ПЗВОС, 15+ ЗЭП, 10+ ПДВ/ПДО одновременно (2026 год)
- **Подрядные организации**: Профстрой, Стандарт Эко, Экодокс, Проект Сервис, Таракиёт
- **Архив**: выполненные работы с 2017 года

### 1.4 Типы объектов заказчиков

| Категория | Примеры |
|---|---|
| **Добыча и переработка** | Карьеры (гранит, мрамор, гипс, кварц), дробилки, обогатительные фабрики |
| **Промышленность** | Цементные заводы, кирпичные заводы, ЖБИ, металлообработка, химпроизводство |
| **Энергетика** | ТЭС, АБЗ (асфальтобетонные заводы), нефтебазы, АЗС/АГЗС |
| **Агропром** | Птицефабрики, маслозаводы, молочные хозяйства, теплицы |
| **Сервис (4-тоифа)** | Гостиницы, рестораны, магазины, автосервис, клиники, аптеки |
| **ГРР** | Месторождения урана, золота, фосфоритов, строительных материалов |
| **Инфраструктура** | Водопроводы, полигоны ТБО, очистные сооружения |

## 3. Типы выполняемых работ

### 3.1 ПЗВОС — Проект заявления о воздействии на окружающую среду

**Когда требуется**: при строительстве новых объектов, реконструкции, расширении производства — если установлен факт воздействия на окружающую среду и здоровье граждан.

**Состав документации ПЗВОС**:

| Раздел | Содержание |
|---|---|
| Титульный лист | Наименование объекта, заказчик, разработчик |
| Общие сведения | Описание предприятия, местоположение, координаты |
| Характеристика района | Климат, рельеф, гидрология, флора/фауна |
| Технологическая часть | Описание технологии, оборудование, сырьё |
| Воздействие на атмосферу | Инвентаризация источников, расчёт рассеивания (ОНД-86), ПДВ |
| Воздействие на водные ресурсы | Баланс водопотребления, сбросы, ПДС |
| Отходы производства | Инвентаризация отходов, классы опасности, ПДО |
| Мероприятия по охране | Предлагаемые меры снижения воздействия |
| Жамоат эшитуви | Протокол общественных слушаний |
| Заключение (Хулоса) | Итоговая оценка допустимости воздействия |
| Приложения (Иловалар) | Карты, схемы, расчёты, согласования |

**Сопроводительные документы**:
- Карта рассеивания (.cdr → .pdf) — визуализация распространения ЗВ
- Ситуационный план (.cdr → .pdf) — расположение объекта
- Баланс водопотребления (.xlsx)
- Таблица исходных данных (.doc)
- Координаты объекта (.docx)
- Спутниковые снимки (.jpg, .kmz)
- Цифровая подпись (.pfx)

**Средний объём**: основной документ 20+ МБ (.doc), финальная сборка 50–100 страниц (.pdf)

### 3.2 ПДВ — Предельно допустимые выбросы

**Состав**:
- Инвентаризация выбросов (.docx) — перечень всех источников и веществ
- Нормативы ПДВ (.docx) — предлагаемые нормативы по каждому веществу
- Расчёт рассеивания (ОНД-86) — подтверждение ненарушения ПДК
- Карта рассеивания (.cdr)
- План-схема выбросов (.pdf)
- Паспорта оборудования (.doc/.pdf)

### 3.3 ПДО — Предельно допустимые объёмы отходов

**Состав**:
- Инвентаризация отходов (.docx) — классификация по видам и классам опасности
- Нормативы ПДО (.docx) — лимиты образования и размещения
- Паспорта отходов (.doc)
- Расчёты (.xls)
- План-схема отходов (.pdf)
- Договоры на вывоз/утилизацию (.pdf)

### 3.4 ЗЭП — Заключение экологической экспертизы проекта

**Комплексный документ**, синтезирующий результаты ПЗВОС, ПДВ, ПДО и дополнительного анализа.

**Состав**:
- Основной отчёт (800+ КБ, .docx)
- Содержание/структура
- Акт обследования (.docx)
- Лимиты (.xls)
- Приложения — источники выбросов (.docx)
- Все карты и схемы из ПДВ/ПДО
- Ссылки на ПЗВОС

### 3.5 КЭН — Корпоративные экологические нормативы

Разработка экологических нормативов для действующих предприятий. Стоимость проектов: 4.5–5.5 млн сум.

### 3.6 РСВ — Расчёт степени воздействия

Утверждённые формы оценки экологического воздействия. Используются стандартизированные шаблоны.

### 4.2 Детализация этапов

**Этап 1–2: Приём заявки и вопросник**
- Клиент обращается с запросом
- Заполняется вопросник (шаблоны в `Вопросники для ПДО,ПДВ,ПДС,ЗВОС и ЗЭП/`)
- Собираются: кадастровые документы, координаты, тех.паспорта оборудования, договоры

**Этап 3–4: Коммерческое предложение и договор**
- КП формируется в Word (.docx), подписывается в PDF
- Структура: по подрядчику → по номеру → по типу работ
- Сметы в Excel (.xls)
- Хранение: `Коммерч. предложения и сметы/КП {ГОД} год БРР/`

**Этап 5–6: Сбор данных и разработка**
- Основной документ пишется в Word (.doc/.docx)
- Данные по оборудованию, веществам, объёмам вносятся вручную
- Баланс водопотребления рассчитывается в Excel

**Этап 7: Расчёты рассеивания**
- **Текущий инструмент**: ond86.exe (десктопная программа, файлы .ond)
- **Новый инструмент**: веб-приложение ОНД-86 (в разработке)
- Результаты переносятся в текст документа вручную

**Этап 8: Карты и схемы**
- Карта рассеивания — CorelDRAW (.cdr)
- Ситуационный план — CorelDRAW (.cdr)
- План-схема выбросов/отходов — CorelDRAW (.cdr)
- Спутниковые снимки — Google Earth (.kmz, .jpg)
- Экспорт в PDF для включения в документ

**Этап 9–10: Сборка и подпись**
- Все документы собираются в единый PDF (СОБР.)
- Подписываются с использованием цифровой подписи (.pfx)
- Протоколы общественных слушаний прикладываются

**Этап 11–13: Экспертиза и доработка**
- Документ подаётся на государственную экологическую экспертизу
- При замечаниях — папка «ДОРАБОТКА» / «Қайта ишлаш учун»
- После устранения замечаний — повторная подача
- Получение положительного заключения

### 4.3 Типичные сроки

| Тип работы | Срок | Стоимость (ориентир) |
|---|---|---|
| ПЗВОС (простой объект, 4-тоифа) | 15–20 рабочих дней | 5–15 млн сум |

## 9. Существующий модуль ОНД-86 (веб-приложение)

### 9.1 Общее описание

Веб-приложение для расчёта рассеивания ЗВ в атмосфере по методике ОНД-86, заменяющее десктопную программу `ond86.exe` и ручную визуализацию в CorelDRAW.

**Расположение**: `C:\Users\TEXNO\Documents\Projects\My first project\`

### 9.2 Технический стек

| Компонент | Технология |
|---|---|
| **Frontend** | React 18.3.1, Vite 5.3.4, Leaflet 1.9.4 |
| **Backend** | Python 3.13, FastAPI 0.135.1, Uvicorn |
| **Расчёты** | NumPy 2.4.3 (векторизованные вычисления) |
| **Визуализация** | Leaflet + Canvas heatmap, Matplotlib |
| **Экспорт** | ReportLab (PDF), OpenPyXL (Excel) |
| **Метеоданные** | Open-Meteo API (бесплатное) |
| **Языки UI** | Русский / Узбекский (i18n.js) |

### 9.3 Реализованный функционал

**Ввод данных:**
- Источники выбросов: карточки (1–10), таблица (1–50), импорт CSV/Excel
- Метеорология: выбор города, ручной ввод, загрузка из Open-Meteo API
- Вещества: 12 встроенных + пользовательские с указанием ПДК

**Расчёт:**
- Ядро ОНД-86: горячие и холодные источники

- 36 направлений ветра для наихудшего случая
- Коэффициенты устойчивости по Пасквиллу (A–F)

**Визуализация:**
- Тепловая карта (heatmap) на Leaflet
- Сетка с числовыми значениями

## 10. Проблемы текущего процесса

### 10.1 Управление проектами

| Проблема | Последствия |
|---|---|
| Статусы проектов отслеживаются в Excel | Нет единого актуального представления, данные устаревают |
| Нет централизованного трекинга сроков | Риск пропуска дедлайнов, сложно планировать загрузку |
| Руководитель (CEO) не видит общую картину в реальном времени | Ручные отчёты от сотрудников, потеря времени |
| Нет контроля этапов (вопросник → КП → разработка → экспертиза) | Непрозрачность: на каком этапе каждый проект |
| Финансовый учёт в разрозненных Excel-файлах | Сложно контролировать дебиторку и рентабельность |

### 10.2 Разработка документации

| Проблема | Последствия |
|---|---|
| Ручной перенос данных из расчётов в Word | Ошибки, потеря времени |
| Каждый документ создаётся с нуля по образцу | Нет единого генератора из шаблонов |
| Карты рисуются вручную в CorelDRAW | Долго, требует специальных навыков |
| Вещества и ПДК ищутся в справочниках вручную | Риск использования устаревших данных |
| Нет версионирования документов | Сложно отследить изменения, риск потери данных |

### 10.3 Коммуникация и хранение

| Проблема | Последствия |
|---|---|
| Сетевой диск без структурированного доступа | Любой может изменить/удалить файлы |
| Нет уведомлений о смене статуса проекта | Ручное информирование |
| Нет единой базы контактов заказчиков | Информация разбросана по папкам |
| Дубликаты ОНД-86 Калькулятора в разных папках | Несогласованность версий |
